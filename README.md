## 🎯 評分系統

所有餐廳採用 **50 分制評分系統**：

| 評分項目 | 分數範圍 | 說明 |
|---------|---------|------|
| 口味品質 | 0-10 | 食物品質、道地性、料理技法 |
| 價值性價比 | 0-10 | 價格合理度、份量 |
| 便利性 | 0-10 | 地點、訂位難易度、營業時間 |
| 穩定性 | 0-10 | 評論一致性、可靠度 |
| 風險控制 | 0-10 | 踩雷機率（10分 = 低風險）|

### 評分參考

- **40+ 分**：極力推薦 🌟
- **35-39 分**：非常好的選擇 ⭐
- **30-34 分**：可接受的備選
- **<30 分**：考慮排除

> 📖 詳細評分標準與研究方法 → [AGENTS.md](AGENTS.md)
