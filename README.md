## 評分標準

所有餐廳採用 50 分制評分系統：
- 口味品質（0-10）
- 價值性價比（0-10）
- 便利性（0-10）
- 穩定性（0-10）
- 風險控制（0-10，10 分為低風險）

**評分參考**：
- 40+ 分：極力推薦
- 35-39 分：非常好的選擇
- 30-34 分：可接受的備選
- <30 分：考慮排除
